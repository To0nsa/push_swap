Detailed Description

Assigns normalized indices to stack A values for radix sort.

This file includes functions to:

  • Normalize values of stack A by assigning their sorted index
  • Store original values for restoration after sorting

Used primarily in radix sort to simplify bitwise sorting based on positive index rather than raw value.
