Uses ft_check_overflow to validate input and supports + and - signs. Returns a sentinel value (ATOI_ERROR) on overflow and sets an error flag.

Main argument parsing and validation function.

  • Accepts both quoted strings and multiple arguments
  • Converts input to integers
  • Validates format, uniqueness, and range
  • Sets up array->a and array->a_size

Exits on failure. If the array is already sorted, the program exits immediately to save operations.
