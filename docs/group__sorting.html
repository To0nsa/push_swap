◆ sort_array()

void sort_array ( t_array *  array)

Sorts stack A using the most efficient strategy based on size.

Chooses from different sorting algorithms depending on the number of elements in stack A:

  • 2 → sort_two
  • 3 → sort_three
  • 4–5 → sort_four_and_five
  • 6+ → radix_sort
