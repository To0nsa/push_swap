Detailed Description

Header file for the push_swap project (42).

Defines the core data structure and declares functions used in the push_swap sorting algorithm. The goal of the program is to sort a list of integers using only a limited set of stack operations (push, swap, rotate).

This file includes parsing, error handling, operation implementations, and multiple sorting strategies, including radix sort and small-size optimizations.
