Detailed Description

Validates and parses program arguments into stack A.

Handles both split-string and multiple-argument formats. Performs strict error checking for invalid characters, overflows, duplicates, and sorts.

Converts ASCII strings into integers using a custom ft_atoi_strict and ensures uniqueness and valid bounds before initializing stack A.
